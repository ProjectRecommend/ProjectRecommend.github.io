Project Recommend

An Offline Music Recommender

About Project

Recommend is a project that uses clustering on metadata(ID3-tags) of your music files to Recommend you similar songs from your Offline Music library

Goal of this project is to provide recommendation on your offline music collection that you have in your computer locally.

For more information see Software Requirements Specification of project
